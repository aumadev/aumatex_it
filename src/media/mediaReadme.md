# Media layout
- `images/raw`: original assets before optimization.
- `images/optimized`: exported, web-ready images referenced in pages.
- `icons/svg`: inlineable icons and logos in SVG.
- `icons/favicons`: favicon set and manifest.
- `video`: mp4/webm clips.
- `audio`: podcast or sfx files.
- `fonts`: custom font files.
- `downloads`: PDFs or other downloadable assets.
